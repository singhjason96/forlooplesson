Life Without For Loops!!

Let's count to from 1 to 10

Hover Here

For Loop Syntax

for(intializer; conditional; incrementor){

}

The Initializer

let i = 1

We are making a variable called i and we are going to start it at 1.

We are telling the loop where to start counting and we want to start counting at 1 so we set i to 1

The Conditional

i <= 10

This tells the loop where to end and it tells the loop to keep going until the condition is false.

So until i is not equal to or less than 10

The Incrementor

i++

Our incrementor is what iterates the loop. So we set our initializer to zero but now we need to keep increasing i so we can create more iterations.

What i++ does is that it increases i by 1 after each iteration. So i++ = i + 1

Putting It All Together!!!

for(let i = 1; i<= 10; i++){

}

We set our initializer to 1 and we have our incrementor increasing our initializer by 1 which is why we get every number after 1. We're telling our loop to iterate. Then our conditional is making sure that after every iteration that each number is either less then or equal to 10. After i = 10 our incrementor i++ increases 10 by 1 making i = 11 but our conditional looks at i and understands that 11 is not less than or equal to 10 and so the loop stops.

i = i + 1; // i = 1 + 1 // i = 2

i = i + 1; // i = 2 + 1 // i = 3

i = i + 1; // i = 3 + 1 // i = 4

i = i + 1; // i = 4 + 1 // i = 5

i = i + 1; // i = 5 + 1 // i = 6

...

i = i + 1; // i = 9 + 1 // i = 10

i = i + 1; // i = 10 + 1 // i = 11

Our conditional i <= 10 checks to see if i

is less than or equal to 10. Since 11 is not less

than or equal to 10

The loop stops.

Excercise!!!

Let's create a for loop that adds up all the numbers from 1 to 15.

1 + 2 + 3......+ 15

Solution

let sum = 0;

// This is going to collect our numbers

for (let i = 1; i <= 15; i++) {

sum = sum + i;

// we're taking our sum that starts at 0 and we're adding each value of i to it

//so after each iteration the sum is changed

}

console.log(sum);

//when the loop is done it displays the new sum amount.
